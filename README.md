#  CyberGuardSA - South African Cybersecurity Awareness Assistant

A C# console application that educates South African citizens about cybersecurity threats through interactive conversation, voice greetings, and visual elements.

## Project Overview

CyberGuardSA is a cybersecurity awareness chatbot developed for The IIE's Portfolio of Evidence (POE) project. It addresses the growing cyber threats in South Africa by providing an engaging, educational experience that teaches users about:
- Password safety and management
- Phishing detection and prevention
- Secure browsing habits
- Two-factor authentication
- Malware protection
- Social media privacy
- Wi-Fi security
- Ransomware prevention

##  Features

-  **Voice Greeting** - AI-generated voice welcome on startup
-  **ASCII Art Logo** - Visual branding with cybersecurity theme
-  **Personalized Conversations** - Uses user's name throughout interaction
-  **Colorful Console UI** - Enhanced visual experience with colored text
-  **Typing Effect** - Simulates natural conversation flow
-  **Cybersecurity Education** - Comprehensive responses on 8+ topics
-  **Input Validation** - Handles empty and invalid inputs 
- **Modular Code Structure** - Organized into separate classes
-  **GitHub Actions CI** - Automated builds and testing
